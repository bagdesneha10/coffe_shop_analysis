# coffe_shop_analysis
Coffee Shop Sales Analysis using Python

 Project Overview
This project analyzes a Coffee Shop Sales dataset using Python, Pandas, Matplotlib, and Seaborn. The objective is to discover sales trends, customer purchasing behavior, product performance, and store-wise insights through data analysis and visualizations.

 Objectives
- Analyze overall coffee shop sales performance.
- Compare sales across different store locations.
- Identify top-selling product categories and product types.
- Study monthly and weekday sales trends.
- Visualize data using different charts for better understanding.

 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Analysis Performed
- Data loading and exploration
- Descriptive statistics
- Correlation analysis
- GroupBy operations
- Pivot Tables
- Store-wise sales analysis
- Product category and product type analysis
- Month-wise and weekday-wise sales analysis
- Interactive user input for store and product filtering

Visualizations
-  Bar Charts
- Line Charts
-  Pie Charts
-  Histograms
-  Scatter Plot
-  Pair Plot
-  Violin Plot
-  Swarm Plot
-  Strip Plot
-  Joint Plot
-  KDE Plot
-  Count Plot

Key Insights
- Different store locations contribute differently to total revenue.
- A few product categories generate the majority of sales.
- Some product types are purchased significantly more than others.
- Sales vary across weekdays and months.
- Visualizations help identify trends, distribution, and relationships among sales variables.

 How to Run
1. Clone this repository.
2. Install the required libraries:
   pip install pandas matplotlib seaborn
3. Place the dataset inside the "DataSets" folder.
4. Run the Python script:
   python coffee_shop_sales_analysis.py

 Dataset
File: "CoffeeShopSales-cleaned (1).csv"
Main columns used:
- Transaction Date
- Store Location
- Product Category
- Product Type
- Unit Price
- Transaction Quantity
- Total Amount
- Weekday
